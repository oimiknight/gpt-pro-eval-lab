# Roadmap

GPT Pro Eval Lab is an early public project for repeatable evaluation of AI coding tools in open-source maintenance workflows.

## v0.1.0 Initial Public Version

- Static browser app for scenario selection and scoring
- Five initial evaluation scenarios
- Five-part scoring rubric
- Local-only notes storage
- JSON report export

## v0.2.0 Maintainer Workflow Scenarios

- Add issue triage scenarios for duplicate detection, severity labeling, and maintainer response drafting
- Add pull request review scenarios for correctness, test coverage, and regression risk
- Add release workflow scenarios for changelog drafting and release readiness checks

## v0.3.0 Reproducible Evaluation Packs

- Add scenario metadata for expected model capabilities and failure modes
- Add sample reports for multiple model runs
- Add import support for saved reports
- Document a repeatable comparison workflow for maintainers

## v0.4.0 Project Collaboration

- Add issue templates for new scenario proposals
- Add contribution guidelines for adding benchmark scenarios
- Add a public results format that can be referenced from GitHub issues and discussions

## Open Questions

- Which maintainer workflows should become first-class categories?
- How should reports compare multiple model runs without implying a universal benchmark score?
- Which fields should be included in exported reports while keeping them easy to review?
